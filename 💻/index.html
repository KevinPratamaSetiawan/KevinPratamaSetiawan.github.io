                  Make Schedule item
                  
                            
                  
                            
                  
                                
                  
                                    
                                        
                                    
                                    
                    
                                        
                  • To make schedule item, use [*any] to make any kind of schedule item label.
                    • 
                                        
                    • 
                                        
                  • Here some pre-established label that commonly used:
                    • 
                                        
                  • 
[S]: for Scheduled schedule item.
                    • 
                                        
                  • 
[D]: for Daily schedule item.
                    • 
                                        
                  • 
[W]: for Weekly schedule item.
                    • 
                                        
                  • 
[A]: for Assignment schedule item.
                    • 

                  • 
                                        
                    • 

                    • 
                                        
                  • Additionally there is added label after the main label to give more context, like:
                    • 
                                        
                  • 
-T]: for Today schedule item that has a day name that same with current date day.
                    • 
                                        
                  • 
-TDY]: for Today Date schedule item that has a date that same with current date.
                    • 
                                        
                  • 
-TMW]: for Tomorrow schedule item that has a date that next with current date.
                    • 
                                        
                    • 
                                        
                  • To have context label you need to add either day name, date in dd:mm:yy format, or both on title part.
                    •

                    • 
                                        
                  •

                    • 
                                        
                  • 
-...: for - list level 3.
                    • 
                                        
                  • 
+.: for + list level 1.
                    • 
                                        
                  • 
+..: for + list level 2.
                    • 
                                        
                  • 
+...: for + list level 3.
                    • 
                                    
                  
                                
                  
                                T-003

                • 
                            
                  
                                
                  List style 2 |  & 
                  
                            
                  
                            
                  
                                
                  
                                    
                                        
                                    
                                    
                    
                                        
                  • To make list style 2, use *.:
                    • 
                                        
                  • 
*.: for * list level 1.
                    • 
                                        
                  • 
*..: for * list level 2.
                    • 
                                        
                  • 
*...: for * list level 3.
                    • 
                                        
                  • 
@.: for @ list level 1.
                    • 
                                        
                  • 
@..: for @ list level 2.
                    • 
                                        
                  • 
@...: for @ list level 3.

                    • 
                                        
                  • 
=.: for = list level 1.
                    • 
                                        
                  • 
=..: for = list level 2.
                    • 
                                        
                  • 
=...: for = list level 3.
                    • 
                                        
                  • 
%.: for % list level 1.
                    • 
                                        
                  • 
%..: for % list level 2.
                    • 
                                        
                  • 
%...: for % list level 3.
